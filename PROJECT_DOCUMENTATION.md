## ğŸ¯ Project Overview

### What is CareerPath?

CareerPath is a comprehensive career guidance platform that empowers students and professionals to unlock their potential through:
- **AI-driven career assessments** using GPT-4
- **One-on-one mentorship** from industry leaders
- **Comprehensive job marketplace** with smart matching
- **Personalized career roadmaps** generated by AI
- **Payment integration** for premium mentor sessions

### Target Users

1. **Students** - Career exploration, skill development, job search
2. **Professionals** - Career transitions, upskilling, networking
3. **Mentors** - Share expertise, earn income, build reputation
4. **Companies** - Post jobs, find talent, employer branding

### Key Value Propositions

âœ… **Personalized Guidance** - AI-powered assessments tailored to individual strengths  
âœ… **Expert Mentorship** - Connect with verified industry professionals  
âœ… **Career Roadmaps** - Step-by-step plans generated by AI

```sql
CREATE TABLE users (
  id UUID PRIMARY KEY,
  email VARCHAR(255) UNIQUE NOT NULL,
  full_name VARCHAR(255),
  avatar_url TEXT,
  phone VARCHAR(20),
  bio TEXT,
  skills TEXT[],
  interests TEXT[],
  education JSONB,
  experience JSONB,
  role VARCHAR(50) DEFAULT 'student',
  created_at TIMESTAMPTZ,
  updated_at TIMESTAMPTZ
);
```

#### 2. **mentors** - Mentor Information
```sql
CREATE TABLE mentors (
  id UUID PRIMARY KEY,
  user_id UUID REFERENCES users(id),
  expertise TEXT[],
  hourly_rate DECIMAL(10,2),
  availability JSONB,
  rating DECIMAL(3,2),
  total_sessions INTEGER,
  verified BOOLEAN,
  is_near_peer BOOLEAN,
  linkedin_url VARCHAR(500),
  github_url VARCHAR(500),
  created_at TIMESTAMPTZ,
  updated_at TIMESTAMPTZ
);
```
